# php2html.js
Compile PHP to a simple HTML file.
# Usage
To add php2html.js to your website, add this:

```
<script src="src/php2html.js"></script>
```
To use php2html.js, use this simple script:

```
php2html.loadUrl('yourphpfile.php');
```
